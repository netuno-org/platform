# Setup

The schema of data structure is defined here.

The goal is to ensure that all data is correctly prepared before the application runs.

When the application runs within an empty database, it will be loaded as is it is defined here.

The data structure, such as tables and columns, will be auto-generated here as you build it in the 
interface. They will be created or added in other database environments when the application starts.

You can also program the initial data, which will also be loaded when the application starts.

For more information, please check [the Data Setup tutorial here](https://doc.netuno.org/docs/academy/server/setup).
