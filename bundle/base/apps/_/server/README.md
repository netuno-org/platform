# Server

Here is Netuno's Platform backend.

The polyglot scripting framework makes the REST API development easy and fast.

The supported languages with their code file extensions are:

- JavaScript`.js`
- Python`.py`
- Ruby`.rb`
- Kotlin`.kts`
- Groovy`.groovy`

The backend folders and their purpose are:

- `actions` - Injects code behavior in form and database modifications, more about [Actions](https://doc.netuno.org/docs/academy/server/actions).
- `components` - Custom form fields and data structures.
- `core` - Scripts of core event cycles and shared scripts used by others, more about [Core Imports](https://doc.netuno.org/docs/academy/server/import).
- `reports` - Reports source code.
- `services` - REST API endpoints source code, more about [REST API](https://doc.netuno.org/docs/academy/server/services/rest).
- `setup` - Database scripts for building and loading the schema, more about [Data Setup](https://doc.netuno.org/docs/academy/server/setup).
- `templates` - Templates source code, usually HTML for page parts injection and e-mail contents, more about [Templates](https://doc.netuno.org/docs/academy/server/templates).
